# Handy Touch PHP Template

## Install
Upload all files to your hosting root (e.g., public_html).

## Pages
- index.php
- about.php
- services.php
- safety-supplies.php
- portfolio.php
- contact.php

## Inquiries
Form posts to contact-submit.php and saves to /storage/inquiries.jsonl (created automatically if permissions allow).

If your host blocks creating folders/files, create a folder named `storage` with permission 755.
